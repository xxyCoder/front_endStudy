# 数据类型
## symbol
    - 是原始值（typeof 返回 symbol），且实例唯一、不可变
    - 确保了对象属性使用唯一标识符，不会发生属性冲突

## 基本用法
    - 使用Symbol() 不带new关键字
    - 可以传入描述符，但是即使symbol的描述符一样，两个也是不相等的
    - 共享和重用
        - Symbol.for(key)
        - 使用字符串会检查注册表是否存在，不存在则创建新符号实例并注册，存在则返回
        - 但是和Symbol()注册的不同
        - Symbol.keyFor() 查询全局注册表
            查询符号实例，返回对应描述符（字符）
